# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately. **Do not open a public GitHub issue.**

- **Contact:** sandeep.mothukuris@gmail.com
- Please include a clear description, steps to reproduce, and impact assessment.
- You will receive an acknowledgement within 72 hours.

## Supported Versions

The latest release on the `main` branch receives security updates.

## Scope

This is a lab environment intended for learning and authorized testing only. Default credentials shipped in `.env` files **must** be changed before any deployment. Never expose lab services to the public internet.
